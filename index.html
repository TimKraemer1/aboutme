Smart Weather Station

Flask, HTML, CSS, Pandas, HTTP, Ajax JS, Arduino
I designed, prototyped, and implemented a smart weather station which collects an array of data from a number of sensors (Temperature, Humidity, Atmospheric Pressure, Luminosity, and Wind Speed) via a micro-controller, and send the data to a single board computer (Raspberry Pi) via serial USB communication for data processing

The data that is collected is then parsed, filtered, and stylized before it is sent via HTTP GET Request to remote web-server hosted by PythonAnywhere for display and viewing on a personalized web-page. New data is sent every 10 seconds for the most accurate results

The web-page was developed using Python's Flask framework, as well as HTML and CSS styling. Dynamic page updates are done using JavaScript Ajax functions that request new json data from the Flask app every 10 seconds.

Twitter Sentiment Stock Analyzer, Stanford Hackathon 2021

Python, Pandas, VADER
This project was developed in conjunction with Jackson Newman (University of California, Santa Barbara) for the 2021 Stanford TreeHacks hackathon. Using VADER sentiment analysis, Pandas, Kaggle datasets, and PyPlot, we were able to show the correlation between a company's historical stock price and the stocks sentiment on Twitter

We developed a formula using Laplace Smoothing that allowed us to weight tweets that received more engagement on Twitter as a higher weight for the overall sentiment calculation.

Overall this was a very fun idea to make a reality. Even though the hackathon was held remotely, we still had a blast coming up with fun and clever solutions to bugs, adding new features to make data presentation better, etc.
